Function Documentation

Detailed documentation of the BAMN Engine's functions

Function: renderSprite

Description: Renders a sprite on the screen using JSON frame data.

void renderSprite(Sprite sprite, int x, int y);

Parameters:
- sprite: The sprite object to render.
- x: The x-coordinate on the screen.
- y: The y-coordinate on the screen.

Function: playSound

Description: Plays a sound effect using SDL Mixer.

void playSound(string soundFile);

Parameters:
- soundFile: The path to the sound file to play.

Function: updateGameObject

Description: Updates the state of a game object based on its components.

void updateGameObject(GameObject obj);

Parameters:
- obj: The game object to update.

Function: handleInput

Description: Processes user input and updates game state accordingly.

void handleInput(InputEvent event);

Parameters:
- event: The input event to process.
